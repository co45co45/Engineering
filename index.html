It's Kubernetes Time!

Here at Hawaiian Airlines we are developing Kubernetes application containers.  This application container runs off Apache Tomcat 8.5 which resides on a Ubuntu platform running on a Docker built image.  This Docker image has been deployed to AWS Kubernetes and can be accessed through multiple containers throughout a cluster.  

 

If one container should fail, another will spin up to take it's place.
